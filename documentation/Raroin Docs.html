Changing Fonts:

In order to use google fonts, you need to visit

Google Fonts Library

and follow the instructions on how to get a new font. After you have decided on the styles of the font that you want to load, or download it in ./assets/fonts/

after you need to add the name you font in./assets/scss/bases/bases.scss like this :

/*--------------------------------------------------------------
## Include Fonts
--------------------------------------------------------------*/
@font-face {
    font-family: "circular";
    src: url("../fonts/circular.ttf") format("truetype");
}
@font-face {
    font-family: "circular_bold";
    src: url("../fonts/Circular-Bold.ttf") format("truetype");
    font-weight: normal;
}

then Go to ./assets/scss/1-helpers/_variables.scss and edit this file with your new font like that:

//variables.scss
$font-primary:"circular";
$font-secondary:"circular_bold";
